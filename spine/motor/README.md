##motor.ino

Connects computer to physical world

###limbs
1. Mains Relays (3)
	- __INPUT__: 'A', 'B', or 'C' via Serial
	- __OUTPUT__: respective relay turns _ON_

	- __INPUT__: 'a', 'b', or 'c' via Serial
	- __OUTPUT__: respective relay turns _OFF_
	
	

###Dependencies
- ino (http://inotool.org/)

###Instructions:
>#in this directory...

>ino build

>ino upload
