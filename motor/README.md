##motor

Connects computer to physical world

###current controls
1. Mains Relays (3)
	- Turn AC circuits on and off
	

###dependencies
- [node-serialport](https://github.com/voodootikigod/node-serialport)
- [socket.io](http://socket.io/)

###install and run
> upload sketch.ino to arduino

> npm install

> node motor.js
