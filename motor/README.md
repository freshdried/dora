##motor

Connects computer to physical world

###current controls
1. Mains Relays (3)
	- Turn AC circuits on and off
	

###dependencies
- [node-serialport](https://github.com/voodootikigod/node-serialport)

###install and run
> upload motor.ino to arduino

> npm install serialport

> node motor.js
